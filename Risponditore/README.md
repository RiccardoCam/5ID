La classe Server si mette in contatto con i clients e li mette in contatto con la classe Cinema.
La classe Client riceve le domande dalla classe Cinema e le risponde.
La classe Cinema è quella che gestisce la conversazione.
Tutte le domande e varie opzioni di risposta sono state create e riposte nella classe Grafo che si trova insieme alla classe Cinema,
il tutto viene gestito dal Cinema che attraverso le risposte del Client, formate dal tipo int si fa dare la domanda successiva dal Grafo e la invia al Client.
